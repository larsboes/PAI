# Coding

> Deep coding knowledge for specific languages, frameworks, and architectural paradigms.

## What's Included

| Skill | Purpose |
|-------|---------|
| architecture | Clean Architecture, DDD, hexagonal, CQRS — structural patterns for production codebases |
| data-engineer | EDA, schema design, dataset generation, data transformation, lineage mapping, validation |
| typescript | TypeScript type patterns, async, tsconfig + Bun runtime/tooling + React component architecture |
| swift | Swift/SwiftUI, macOS development, concurrency, plugin architectures |

| fluentbit | FluentBit configs, Lua filters, CI testing patterns |

## Design Principle

These skills share a common trait: **deep reference knowledge for writing code in a specific stack/paradigm.** They're not generic workflow tools — they encode the patterns, gotchas, and best practices you need when writing code in that world.
