# Trip Planning Checklists

Actionable checklists with real deadlines.

## Pre-Trip Checklist (8 weeks out)

- [ ] **Confirm dates** (with yourself and with anyone you're going with)
- [ ] **Check passport expiration** — needs 6+ months validity for EU
- [ ] **Book flights/train** (prices typically best 6-8 weeks out)
- [ ] **Reserve accommodation** (good places book fast)
- [ ] **Add to calendar** (flights, accommodation check-in, key events)

## Bookings Checklist (6-4 weeks out)

- [ ] **Confirm all transport** (have confirmation numbers)
- [ ] **Confirm accommodation** (have confirmation, cancellation policy)
- [ ] **Register for any events** (conferences, meetups, specific experiences)
- [ ] **Check visa requirements** (if applicable)
- [ ] **Notify employer/clients** of travel dates
- [ ] **Block calendar** at work (vacation, limited availability, timezone)

## Work Setup Checklist (2 weeks out)

- [ ] **Define work hours** during trip (are you taking time off or working?)
- [ ] **Test timezone** — what hours can you actually work?
- [ ] **Arrange coverage** if you're unavailable
- [ ] **Download offline maps** (Google Maps offline)
- [ ] **Download translation app** (if needed)
- [ ] **Notify team/clients** of any delayed responses

## Packing Checklist (1 week out)

**Essentials:**
- [ ] Passport + copies
- [ ] Credit cards + some cash
- [ ] Phone charger + adapter (voltage/plugs for region)
- [ ] Medications
- [ ] Laptop + charger (if working)

**Clothing (adjust for season):**
- [ ] Underwear (7 days)
- [ ] Socks (4 pairs)
- [ ] Jeans/work pants (2)
- [ ] T-shirts/tops (3-4)
- [ ] One nice outfit (for meals/events)
- [ ] Jacket/sweater
- [ ] Comfortable shoes + one nicer pair
- [ ] Sleepwear

**Toiletries:**
- [ ] Toothbrush + toothpaste
- [ ] Deodorant
- [ ] Shampoo/soap (or buy there)
- [ ] Medications
- [ ] Contacts/glasses
- [ ] Skincare

**Electronics:**
- [ ] Phone + charger
- [ ] Laptop (if needed)
- [ ] Headphones
- [ ] Power bank

## Day Before Departure

- [ ] Confirm all reservations (check confirmation emails)
- [ ] Check flight/train status (any delays/changes?)
- [ ] Know how you're getting to airport/station
- [ ] Set alarms/reminders
- [ ] Leave out-of-office message if working
- [ ] Lock windows, turn off unnecessary appliances
- [ ] Tell someone your itinerary (safety)

## Return Checklist

- [ ] File any expenses for reimbursement
- [ ] Add memories to vault (trip notes, photos, reflections)
- [ ] Thank hosts/people you stayed with
- [ ] Process any work that accumulated
- [ ] Update Focus areas with learnings/connections made

## Timeline Template

```markdown
- [ ] Confirm dates
- [ ] Book flights/trains
- [ ] Reserve accommodation

## 6 weeks before [DATE]
- [ ] Confirm all bookings
- [ ] Register for events
- [ ] Check visa requirements

## 4 weeks before [DATE]
- [ ] Notify employer/clients
- [ ] Block work calendar
- [ ] Finalize itinerary

## 2 weeks before [DATE]
- [ ] Define work setup
- [ ] Download offline maps
- [ ] Arrange coverage

## 1 week before [DATE]
- [ ] Pack
- [ ] Confirm all reservations
- [ ] Set alarms

## Day of travel
- [ ] Confirm no delays
- [ ] Leave house on time
- [ ] Have confirmation numbers handy

## During trip
- [ ] Check in to accommodation
- [ ] Test wifi
- [ ] Explore enough to feel grounded

## Return
- [ ] File expenses
- [ ] Add to vault
- [ ] Thank people
- [ ] Process work backlog
```

## Why Real Deadlines Matter

❌ "Book train soon" — too vague, you'll procrastinate
✅ "Book train by Jan 15" — specific, trackable

❌ "Pack sometime before departure" — last-minute stress
✅ "Pack Jan 20" — done early, no rushing

For each checklist item that requires action, add a real date.
