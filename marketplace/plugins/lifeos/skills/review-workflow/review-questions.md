# Review Questions & Data Sources

Period-specific prompts and data gathering instructions for each review type.

## Weekly Review

### Data to Gather
- Read all daily notes from the past 7 days (`${VAULT_DAILY_NOTES}/`)
- Check `google-calendar/list-events` for past week's events
- Read any event notes from this week (`${VAULT_EVENTS}/`)

### Reflection Prompts
**Execution:**
- What got done this week?
- What didn't get done and why?
- What tasks carry forward to next week?

**Learning:**
- What did I learn this week?
- Any new skills practiced or knowledge gained?

**Patterns:**
- Which behavioral patterns showed up? (Check against `${PROGRESSIVE_PATTERNS}`)
- Did I practice any counter-strategies? How did they go?

**People:**
- Who did I interact with meaningfully?
- Any relationships that need attention?

**Energy:**
- When did I feel most energized vs. drained?
- What does that tell me about next week's scheduling?

**Next Week:**
- Top 3 priorities for next week
- Any deadlines or events to prepare for?
- One pattern to actively practice

---

## Monthly Review

### Data to Gather
- Read all weekly reviews from the past month (`${VAULT_WEEKLY_REVIEWS}/`)
- Read `${PROGRESSIVE_CURRENT}` for active themes
- Check project progress: read active projects in `${VAULT_PROJECTS}/`
- Read each focus area note (`${VAULT_FOCUS}/`) for current status

### Reflection Prompts
**Achievement:**
- What were the biggest wins this month?
- What projects moved forward? Which ones stalled?

**Focus Balance:**
Run a focus-alignment balance assessment:
- Rate each focus area (1-5) on investment this month
- Which focuses are over-served? Under-served?
- Are current priorities still right?

**Patterns:**
- What behavioral patterns dominated this month?
- Any progress on growth edges? Evidence?
- New patterns noticed?

**Relationships:**
- Key relationship developments?
- Anyone I've been neglecting?

**Direction:**
- Am I moving toward my goals or drifting?
- What should change next month?
- Any commitments to drop or add?

---

## Yearly Review

### Data to Gather
- Read all monthly reviews from the year (`${VAULT_MONTHLY_REVIEWS}/`)
- Read all project notes — categorize: completed, abandoned, ongoing (`${VAULT_PROJECTS}/`)
- Read all focus area notes for deep assessment (`${VAULT_FOCUS}/`)
- Read your personal identity note for identity evolution
- Read key reflection notes for growth evidence (`${VAULT_REFLECTIONS}/`)

### Reflection Prompts
**Big Picture:**
- What am I most proud of this year?
- What would I do differently?
- What surprised me about myself?

**Growth:**
- What skills did I develop?
- Which patterns improved? Which ones persisted?
- How did my identity evolve? (Compare to start of year)

**Relationships:**
- Strongest connections this year?
- Lost connections?
- How did my network change?

**Focus Areas (Deep Review):**
- For each focus: Where was I a year ago? Where am I now? Where do I want to be?

**Next Year:**
- What are my themes for next year?
- What do I want to be true by this time next year?
- Top 3 commitments for the year
- What am I deliberately saying no to?
